# FizzBuzz-algorithm

The code defines a function called fizzBuzz() that asks the user to enter a number. The function then checks whether the number is divisible by 3, 5, or both.
If the number is divisible by both 3 and 5, it returns "FizzBuzz".
If the number is divisible by 3 only, it returns "Fizz".
If the number is divisible by 5 only, it returns "Buzz".
If the number is not divisible by 3 or 5, it returns the number itself.
